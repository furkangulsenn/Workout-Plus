Workout Plus is a basic workout management application where users can record and manage their exercises. It was developed using Windows Forms and uses a file-based storage system.

## 🚀 Features

- Add exercises
- List exercises
- Save to file and read from file
- Simple user interface

## 🛠 Technologies Used

- C#
- Windows Forms (.NET Framework)
- File-based data storage
- Visual Studio 2022

## 📌 Project Status

This is the beginner-level v1 version. New features and improvements will be added in v2.
